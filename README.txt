Краткая инструкция для запуска приложений.

1) Скачать и открыть папку server в IDE
2) В терминале ввести npm i
3) В объекте Client, на строке 7, ввести логин и пароль от postgresql, так же можно указать имя любой существующей пустой базы данных
4) В том же терминале ввести: npm start
5) Скачать и открыть папку client IDE
6) В терминале ввести: npm i
7) В терминале ввести: npm start
